# Notes

This project started as a notebook exploration of hybrid restaurant recommendation approaches.

In this repository:

- the pipeline is implemented as a small Python package (`src/restaurant_recsys/`)
- a compact sample dataset is included for fast reproducibility
- outputs are exported to `outputs/` (metrics, plots, logs, recommendation JSON)
- the legacy notebook prototype is kept under `notebooks/` for reference
- dataset details are documented in `docs/DATASET_LINK.md`
